# Backpack

This Plugin is coded for Multi-Platform Server in mind, that can't provide Client side Mod Support 
with Fabric, Forge or any other Mod-Loader.

At the moment there are the Basic Backpack and 5 Upgraded versions.

- Leather - 9 Slots
- Copper - 18 Slots
- Gold - 27 Slots
- Iron - 36 Slots
- Diamond - 45 Slots
- Netherite - 54 Slots
